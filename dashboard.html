🔧 Firebase 연동 설정

방문자 데이터를 조회하려면 Firebase Realtime Database URL이 필요합니다.
이미 설정된 경우 아래 URL을 확인 후 연결하기를 누르세요.
처음 설정하려면 아래 단계를 따르세요 (2~3분 소요).

  1. Firebase 콘솔 접속 → 프로젝트 추가
  2. 프로젝트명: tesla-charger-analytics
  3. Google Analytics 사용안함 → 프로젝트 만들기
  4. 좌측메뉴 빌드 → Realtime Database → 데이터베이스 만들기
  5. 위치: 미국 → 테스트 모드에서 시작 → 사용 설정
  6. 표시되는 URL을 아래에 붙여넣기
